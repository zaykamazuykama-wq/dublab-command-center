Платформ тус бүрийн үүрэг

Facebook: - Lead авах гол суваг - Page + personal биш brand-style page - Groups дээр шууд spam хийхгүй, problem/education post TikTok: - Reach авах туршилтын суваг - Hook хүчтэй, 15–25 сек YouTube Shorts: - Search + long-term discovery - Before/after demo, workflow proof Inbox: - DM ирсэн хүн бүр lead tracker рүү орно - Reply draft → follow-up → demo санал

30 хоногийн өсөлт + борлуулалтын төлөвлөгөө

Зорилго: 30 хоногт 1000+ relevant үзэгч, 30+ DM/lead, 5+ paid test order авах. 1-р 7 хоног — Итгэл үүсгэх - Өдөр бүр 2 short: 1 demo, 1 education. - CTA: “DEMO” эсвэл “VIDEO” гэж бич. - Гол KPI: comment, DM, save. 2-р 7 хоног — Problem гаргах - “Танай англи video Монгол хүнд ойлгогдож байна уу?” - “Subtitle хангалтгүй. Voiceover илүү ойр.” - 10 бизнес/page рүү soft DM санал явуулах. - Гол KPI: reply rate. 3-р 7 хоног — Offer гаргах - 30–60 сек demo test package санал болго. - 3 package: Short demo / Video localization / Monthly content reuse. - Гол KPI: test order. 4-р 7 хоног — Proof + давталт - Before/after үр дүнг нууцлалтай харуул. - Customer feedback screenshot/quote. - Best performing content-ийг дахин өөр hook-той гарга.

Борлуулалтын offer

Package 1 — Demo test 30–60 сек video хэсгийг Монгол script + voiceover + caption болгож туршина. Package 2 — Short-form localization 1 video → 3 short clips → Монгол voiceover + caption + CTA. Package 3 — Monthly content reuse Сар бүр 4–8 existing video-г Монгол audience-д зориулсан short package болгоно. Public байршуулалт: “Бид tool зардаггүй.
